# 簡易計時器

## 項目描述

此為一簡易計時器應用程式，以Python與PyQt5開發，提供基本計時功能。

## 功能特點

- 啟動/暫停計時
- 重置計時器
- 顯示時、分、秒
- 簡潔易用介面

## 系統需求

- Windows 10 或以上版本
- 已包含所有必要依賴（打包於 _internal/ 目錄）

## 使用說明

1. 雙擊 `簡易計時器.exe` 啟動應用程式
2. 點擊「開始」按鈕啟動計時
3. 點擊「暫停」按鈕暫停計時
4. 點擊「重置」按鈕清零計時器

## 技術棧

- Python 3.13
- PyQt5
- PyInstaller（用於打包）
